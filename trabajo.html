Infografía: Regresión Lineal para Predecir IMC

1. Importar librerías esenciales

Se importan numpy para manipulación numérica y LinearRegression de sklearn para crear el modelo.

import numpy as np
from sklearn.linear_model import LinearRegression

2. Generar datos aleatorios

Se generan n = 100 muestras de alturas y pesos aleatorios dentro de rangos realistas:

n = 100
alturas = np.random.uniform(1.5, 2.0, n)  # alturas en metros
pesos = np.random.uniform(50, 100, n)     # pesos en kilogramos

3. Calcular el IMC real

El IMC para cada muestra se calcula con la fórmula tradicional:

imc_real = pesos / (alturas ** 2)

4. Preparar datos para el modelo

Se juntan los pesos y alturas como variables independientes (features) X y el IMC real como variable objetivo y:

X = np.column_stack((pesos, alturas))
y = imc_real

5. Crear y entrenar el modelo de regresión lineal

Se instancia el modelo y se entrena con los datos generados:

modelo = LinearRegression()
modelo.fit(X, y)

6. Predecir IMC para nuevos datos

Se generan nuevos valores aleatorios para peso y altura, y se predice el IMC usando el modelo entrenado:

peso_nuevo = np.random.uniform(50, 100)
altura_nueva = np.random.uniform(1.5, 2.0)
X_nuevo = np.array([[peso_nuevo, altura_nueva]])
imc_predicho = modelo.predict(X_nuevo)[0]

7. Mostrar resultados

Se imprimen los valores nuevos de peso y altura, el IMC calculado con fórmula y el IMC predicho por el modelo:

print(f"Datos nuevos -> Peso: {peso_nuevo:.2f} kg, Altura: {altura_nueva:.2f} m")
print(f"IMC real calculado: {peso_nuevo / altura_nueva**2:.2f}")
print(f"IMC predicho por modelo regresión lineal: {imc_predicho:.2f}")

Resumen

Este código ilustra cómo crear datos sintéticos para peso y altura, calcular el IMC real, y luego usar un modelo de regresión lineal para aprender la relación y predecir el IMC basado en nuevas entradas. Es un buen ejercicio para practicar aprendizaje supervisado y entender regresión lineal.
